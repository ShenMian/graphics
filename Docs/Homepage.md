# 主页 {#mainpage}

## 概述

一个简单的现代图形 API 抽象层.  

## 特性

- 跨平台支持.
- 多渲染 API 后端: OpenGL, Vulkan, Direct3D, Metal. (only support OpenGL for now)
- 集成 UI: [Dear ImGui](https://github.com/ocornut/imgui).
