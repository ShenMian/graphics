◆ set_raw_mouse_motion()

void Window::set_raw_mouse_motion ( bool  enable)

设置是否启用原始鼠标运动.

参数
enabletrue 为开启, false 为关闭.

原始鼠标运动更接近鼠标在表面上的实际运动. 它不受应用于桌面光标运动的缩放和加速度的影响. 适合控制 3D 相机.

警告
仅在禁用光标时有效.
